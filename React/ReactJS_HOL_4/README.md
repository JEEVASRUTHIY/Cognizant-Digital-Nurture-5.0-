# Blog App

ReactJS Hands-on Lab 4. The class-based `Posts` component loads blog posts
from JSONPlaceholder in `componentDidMount()` and implements
`componentDidCatch()` for lifecycle error handling.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
