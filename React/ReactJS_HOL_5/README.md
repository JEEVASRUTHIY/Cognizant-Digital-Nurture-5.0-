# Cohorts Tracker

ReactJS Hands-on Lab 5. Cohort cards are styled using
`CohortDetails.module.css`; ongoing cohort titles are green and all other
cohort titles are blue.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
