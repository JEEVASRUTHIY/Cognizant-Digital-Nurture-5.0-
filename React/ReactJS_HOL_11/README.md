# Event Examples App

ReactJS Hands-on Lab 11. Includes increment/decrement handlers, a button that
invokes multiple methods, an argument-based welcome handler, a synthetic click
event, and a Euro-to-Rupees currency converter form.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
