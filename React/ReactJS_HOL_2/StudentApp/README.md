# StudentApp

ReactJS Hands-on Lab 2: Student Management Portal.

## Components

- `Home` displays the Home page welcome message.
- `About` displays the About page welcome message.
- `Contact` displays the Contact page welcome message.

All three class components are rendered by `App`.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
