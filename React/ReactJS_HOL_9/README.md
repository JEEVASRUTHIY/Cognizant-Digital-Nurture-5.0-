# Cricket App

ReactJS Hands-on Lab 9. Demonstrates ES6 `map`, `filter`, arrow functions,
array destructuring, array merging with spread syntax, and conditional
rendering.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
