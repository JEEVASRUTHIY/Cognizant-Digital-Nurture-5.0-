# Score Calculator App

ReactJS Hands-on Lab 3. The `CalculateScore` function component accepts a
student name, school, total marks, and goal, then displays the calculated
average score.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
