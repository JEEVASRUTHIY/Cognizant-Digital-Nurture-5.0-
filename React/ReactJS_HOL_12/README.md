# Ticket Booking App

ReactJS Hands-on Lab 12. Guests can browse flight details, while authenticated
users can book tickets. Login and logout switch between `GuestPage` and
`UserPage` using conditional rendering.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
