# Blogger App

ReactJS Hands-on Lab 13. Renders book, blog, and course lists using components,
keys, `map()`, and several conditional-rendering approaches: `if/else`, a
ternary expression, and a component returning `null`.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test -- --watchAll=false
```

## Production build

```bash
npm run build
```
